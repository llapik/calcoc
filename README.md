# AI PC Repair & Optimizer

Загрузочная USB-флешка (256 ГБ) с локальным AI для диагностики, ремонта, оптимизации и модернизации ПК.

## Возможности

- **Диагностика** — CPU, RAM, GPU, диски (S.M.A.R.T.), материнская плата, установленные ОС
- **Анализ проблем** — системные логи, производительность, вредоносное ПО (ClamAV)
- **AI-советник** — рекомендации на естественном языке через локальные LLM или OpenRouter
- **Автоисправление** — файловые системы, загрузчик, реестр Windows, очистка мусора
- **Безопасный откат** — резервное копирование перед каждым изменением + журнал операций
- **Прогнозирование отказов** — анализ трендов S.M.A.R.T., температур, наработки
- **Рекомендации по апгрейду** — узкие места, совместимые комплектующие, приоритеты
- **Веб-интерфейс** — чат с AI на русском/английском через localhost:8080

## Быстрый старт (разработка)

```bash
bash scripts/setup_env.sh
source .venv/bin/activate
python -m src.core.app
# Откройте http://127.0.0.1:8080
```

## Сборка загрузочного образа

```bash
make build                              # Создать ISO
sudo make install-usb USB_DEV=/dev/sdX  # Записать на USB
```

## AI-бэкенды

| Бэкенд | Описание |
|--------|----------|
| `llama` | Локальный — llama.cpp с GGUF моделями (по умолчанию) |
| `openrouter` | Облачный — OpenRouter API (нужен интернет + API ключ) |
| `none` | Без AI — только правила и скрипты |

Модель выбирается автоматически по объёму доступной ОЗУ (от SmolLM 135M до LLaMA 3 8B).

## Уровни безопасности

| Уровень | Описание |
|---------|----------|
| Зелёный | Только чтение, без подтверждения |
| Жёлтый | Обратимые изменения, нужно подтверждение + бэкап |
| Красный | Рискованные операции, нужен повышенный доступ |
| Чёрный | Заблокировано (BIOS flash и т.п.), только режим эксперта |

## Технологии

- Python 3.10+, Flask, llama-cpp-python
- SQLite (журнал, телеметрия), ChromaDB (RAG)
- ClamAV, smartmontools, dmidecode, e2fsprogs
- GRUB (UEFI + Legacy boot)
